# Customer Churn Prediction Dashboard

An end-to-end machine learning project that predicts customer churn for a telecom/SaaS business, served via a FastAPI backend and visualized through an interactive Streamlit dashboard.

## Project Status
🚧 In active development.

## Tech Stack
- **Language:** Python
- **ML Libraries:** Pandas, NumPy, Scikit-learn, XGBoost
- **Backend:** FastAPI
- **Frontend:** Streamlit
- **Deployment:** Render (API) + Streamlit Community Cloud (Dashboard)

## Project Structure
customer-churn-prediction/
├── backend/        # FastAPI application
├── data/           # Raw and processed datasets
├── frontend/       # Streamlit dashboard
├── models/         # Saved trained models
├── notebooks/      # Exploratory Data Analysis
├── src/            # ML pipeline scripts
└── tests/          # Unit and integration tests
